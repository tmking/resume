Projects

Seagull Testing

I worked with two other developers to rescue an abandoned PHP application designed to provide an interface for students registering for environmental certification courses. We abandoned the buggy PHP codebase and used Sinatra and Sequel to model the customer's existing Postgres database and reimplement the needed functionality.

Jordan Kitt's Music Corporate Website and Weblead Application

I migrated Jordan Kitts' network of PHP websites from a dedicated server to a leaner VPS running Nginx, after which I reimplimented the PHP core of the old website as a Rails application. In addition to functioning as the company's corporate web presence, the application generates leads from various customer contact forms and routes them to one of eleven stores based on zip code geolocation (via Geokit). The leads are stored in a MySQL database to facilitate future analysis and reporting. One of the most challenging aspects of this project was developing database models that would be easily compatible with the company's internal MS-SQL database, which is itself a copy (refreshed nightly) of the legacy database used by their inventory/contact management software.

Jordan Kitt's Music eStore

I wrote a simple PHP shopping cart application in 2007 to enable Jordan Kitt's to experiment in online retail. Payment processing was handled by posting form information to PayPal.
